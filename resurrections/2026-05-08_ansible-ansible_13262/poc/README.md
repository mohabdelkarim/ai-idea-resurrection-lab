# Proof of Concept: feature request: looping over blocks

**Language:** python
**Estimated run time:** < 5 minutes

## Prerequisites

- Python 3.12+, pip install requirements

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## What This Demonstrates

The Ansible community is requesting a feature to loop over a block of tasks, allowing for more dynamic and reusable.
