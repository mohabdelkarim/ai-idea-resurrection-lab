# Proof of Concept: A workflow to periodically check status for author-ready PRs opened by a non-collaborator

**Language:** typescript
**Estimated run time:** < 5 minutes

## Prerequisites

- Node.js 20+, npm install

## How to Run

```bash
npm install
npx ts-node main.ts
```

## What This Demonstrates

A GitHub Actions workflow to periodically check status for author-ready PRs opened by non-collaborators.
