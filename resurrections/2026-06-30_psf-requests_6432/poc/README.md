# Proof of Concept: The latest version of requests (2.29.0) does not support urllib3 2.0.0

**Language:** python
**Estimated run time:** < 5 minutes

## Prerequisites

- Python 3.12+, pip install requirements

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## What This Demonstrates

The requests library does not support urllib3 2.0.0 due to outdated dependency range.
