# Proof of Concept: PEP 703 -- Making the Global Interpreter Lock Optional in CPython

**Language:** python
**Estimated run time:** < 5 minutes

## Prerequisites

- Python 3.12+, pip install requirements

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## What This Demonstrates

PEP 703 aims to make the Global Interpreter Lock optional in CPython for improved performance in multithreaded applications.
