# Proof of Concept: Please, oh please, ask, or at least warn, about overwriting ~/.zshrc on installation

**Language:** python
**Estimated run time:** < 5 minutes

## Prerequisites

- Python 3.12+, pip install requirements

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## What This Demonstrates

Implement a warning system to prevent accidental overwriting of ~/.zshrc during Oh My Zsh installation.
