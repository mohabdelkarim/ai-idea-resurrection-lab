# Proof of Concept: Ability to accept input from Charts, Tables, etc.

**Language:** python
**Estimated run time:** < 5 minutes

## Prerequisites

- Python 3.12+, pip install requirements

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## What This Demonstrates

Streamlit should allow users to interact with output elements like charts and tables to send input back to the Python.
