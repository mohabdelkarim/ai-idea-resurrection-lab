# Proof of Concept: Change Request: Show a reminder when errors are suppressed

**Language:** typescript
**Estimated run time:** < 5 minutes

## Prerequisites

- Node.js 20+, npm install

## How to Run

```bash
npm install
npx ts-node main.ts
```

ESLint should optionally show reminders when errors are suppressed to improve visibility of tech debt.
