## Original Description

It would be nice for Bubble Tea to support the "CSI u" Encoding Specification as outlined in Paul Evans' seminal work ["Fix Keyboard Input on Terminals - Please"](http://www.leonerd.org.uk/hacks/fixterms/).

* [Wezterm](https://wezfurlong.org/wezterm/config/lua/config/enable_csi_u_key_encoding.html)
