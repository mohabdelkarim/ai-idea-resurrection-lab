# Proof of Concept: Rails cashes with Ruby 3.1.0 release

**Language:** python
**Estimated run time:** < 5 minutes

## Prerequisites

- Python 3.12+, pip install requirements

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## What This Demonstrates

Rails crashes during project creation with Ruby 3.1.0 due to compatibility issues.
