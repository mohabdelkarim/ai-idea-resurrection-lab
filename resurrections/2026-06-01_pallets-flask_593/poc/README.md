# Proof of Concept: Nestable blueprints

**Language:** python
**Estimated run time:** < 5 minutes

## Prerequisites

- Python 3.12+, pip install requirements

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## What This Demonstrates

Nestable blueprints allow registering sub-blueprints within a parent blueprint for more modular Flask applications.
