# Proof of Concept: Better <think> block rendering for DeepSeek-R1 and similar

**Language:** typescript
**Estimated run time:** < 5 minutes

## Prerequisites

- Node.js 20+, npm install

## How to Run

```bash
npm install
npx ts-node main.ts
```

## What This Demonstrates

Improve rendering of <think> blocks in LLM output with collapsible and visually distinct design.
