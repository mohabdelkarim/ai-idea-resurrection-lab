# Proof of Concept: Feature Request: Option to not check included files in clang-tidy

**Language:** python
**Estimated run time:** < 5 minutes

## Prerequisites

- Python 3.12+, pip install requirements

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## What This Demonstrates

Add an option to clang-tidy to skip analyzing included files for faster linting.
